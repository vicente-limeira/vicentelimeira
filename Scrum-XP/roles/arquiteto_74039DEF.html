Role: Arquiteto
Profissional responsável por garantir o seguimento das diretrizes de um projeto de desenvolvimento de software como qualidade , contexto organizacional, funcionalidade, usabilidade, performance, desempenho e baixo custo de investimento.
Relationships
Primary Performs
  • Definir arquitetura
  • Definir a visão do sistema
  • Detalhar os requisitos
  • Elaborar backLog do projeto
  • Elaborar BackLog do Sprint
  • Projetar a solução
Additionally Performs
  • Homologar release
  • Liberar release
  • Sprint review
Modifies
  • BackLog do projeto
  • Caso de uso
  • Documento de arquitetura
  • Documento de visão
  • Repositório GIT
